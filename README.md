  # terraform-associate-prep
                                                                                                                                                             
  Study repo for the HashiCorp Terraform Associate (003) certification.                                                                                    

  ## Exam
  - **Issuer:** HashiCorp
  - **Cost:** $70
  - **Format:** 57 questions, 60 minutes, multiple choice
  - **Booking:** portal.certiverse.com
  - **Target date:** March 2026

  ## Topics
  - IaC concepts and Terraform workflow
  - HCL syntax — variables, outputs, locals, expressions
  - Resource lifecycle — create, update, destroy, taint
  - State management — local and remote (S3 backend)
  - Modules — writing, sourcing, versioning
  - Terraform Cloud / Terraform Enterprise basics
  - Providers and the registry

  ## Structure
  /notes        # topic-by-topic study notes
  /labs         # hands-on practice configs
  /mock-exams   # practice questions and answers
